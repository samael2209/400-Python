## 🚀 Cómo Usar

### Clonar el repositorio

```bash
git clone https://github.com/samael2209/400-Python.git
cd 400-Python
```

### Ejecutar un ejercicio

```bash
cd 400-python-ejercicios/01_basico
python Ejercicio-001.py
```
